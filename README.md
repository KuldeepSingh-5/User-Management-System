# 🚀 User Management System (MERN Stack)

## 📖 Project Overview
User Management System is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application helps administrators manage user records efficiently through a clean and responsive interface. It supports complete CRUD operations, secure authentication, advanced search, filtering, sorting, profile image upload, CSV export, and pagination.

This project demonstrates full-stack development skills, REST API integration, MongoDB database management, and responsive frontend design.

---

## ✨ Features

### Authentication
- Secure User Login
- User Registration
- JWT Authentication
- Protected Routes

### User Management
- Add New User
- View User Details
- Edit User Information
- Delete User
- Update User Status (Active/Inactive)

### Search & Filter
- Search Users by Name or Email
- Filter by Gender
- Filter by Status
- Sort Users

### Other Features
- Profile Image Upload
- CSV Export
- Pagination
- Responsive Design
- Error Handling
- Form Validation
